# Todo App Handson

タスク管理アプリケーションのハンズオン実装です。シンプルで使いやすいTodoアプリを、フロントエンド技術を使って構築します。

## 概要

このプロジェクトは、Todoアプリケーションの基本的な実装を学ぶためのハンズオン課題です。
タスクの作成、編集、削除、完了状態の管理などの基本的な機能を実装します。

## 技術スタック

- **HTML** - マークアップ構造の定義
- **CSS** - スタイリングとレスポンシブデザイン
- **JavaScript** - ロジック実装とDOM操作
- **LocalStorage** - タスクデータの永続化

## 機能

- [ ] タスクの作成
- [ ] タスクの表示
- [ ] タスクの編集
- [ ] タスクの削除
- [ ] タスクの完了/未完了切り替え
- [ ] LocalStorageへのデータ保存と読み込み

## インストールと実行

```bash
# リポジトリをクローン
git clone https://github.com/pikawakainfo-crypto/todo-app-handson.git

# ディレクトリに移動
cd todo-app-handson

# ブラウザで index.html を開く
open index.html
```

## プロジェクト構成

```
todo-app-handson/
├── index.html       # メインのHTML
├── styles/

├── js/
│   └── app.js      # メインのJavaScriptロジック

```
